# Tailwind CSS

Framework de estilos 'utility-first' para diseño rápido.

## Por qué en minerOS

Te permite diseñar interfaces profesionales (modo oscuro, grids responsive) sin escribir hojas de estilo separadas. Ideal para prototipar rápido.

## Casos de uso

- Layouts de PhotoMine
- Tarjetas del Dashboard
- Botones interactivos

## Código ejemplo

```html
<div class='bg-slate-900 text-white p-4 rounded-xl hover:bg-slate-800 transition shadow-lg'>
  Panel de Control
</div>
```

## Proyectos que lo usan

- DirectOS v7.0 (100% Tailwind)
- PhotoMine Dashboard
- farmaIA v5.0 (UI completa)
