# HTML5 Semántico

El esqueleto fundamental de la web moderna.

## Por qué en minerOS

Porque minerOS es 'Local First' y web. Un buen HTML semántico (`<article>`, `<section>`) hace que tus datos sean portables y accesibles sin depender de frameworks.

## Casos de uso

- Estructura de dashboards
- Reportes generados por IA
- Interfaz de PhotoMine

## Código ejemplo

```html
<article class='card'>
  <header>
    <h2>Factura #102</h2>
  </header>
  <p>Contenido...</p>
</article>
```

## Proyectos que lo usan

- DirectOS v7.0 (estructura completa)
- PhotoMine Dashboard
- farmaIA v5.0 (interfaz web)
