# Git

Sistema de control de versiones.

## Por qué en minerOS

Tu red de seguridad. Te permite experimentar sin miedo a romper nada, pudiendo volver atrás en el tiempo.

## Casos de uso

- Guardar progreso
- Probar features nuevas (ramas)
- Backup en GitHub

## Código ejemplo

```bash
git init
git add .
git commit -m 'Añadido nuevo parser'
```

## Proyectos que lo usan

- Todos los proyectos minerOS (100%)
- PhotoMine v1.4 (45 commits)
- DirectOS v7.0 (branch feature/markdown-content)
