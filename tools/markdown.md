# Markdown

Formato de texto plano para notas y documentación.

## Por qué en minerOS

Es el formato universal de conocimiento. Compatible con Obsidian, GitHub, y fácil de procesar con IA.

## Casos de uso

- Notas de aprendizaje
- Documentación de proyectos
- Knowledge Base

## Código ejemplo

```markdown
# Título
## Subtítulo
- Lista de items
- **Negrita** y *cursiva*

\`\`\`python
codigo()
\`\`\`
```

## Proyectos que lo usan

- knowledge-base.md (1,500+ líneas)
- Todos los README.md de proyectos
- DirectOS v7.0 (content system)
