# Music Theory Lab

9 simulaciones interactivas de teoria musical basada en la proporcion aurea (Golden Harmony).

## Stack
- Canvas 2D para visualizaciones
- Web Audio API para sintesis y reproduccion
- golden-harmony-engine.js (motor compartido)
- GitHub Pages (gh-pages branch)

## Contenido
- **3 Escalas**: 12-phiW, 15-notas, cromatica aurea
- **3 Armonizadores**: acordes y progresiones por sistema
- **3 Compositores**: generacion automatica por sistema
- **Paper**: Golden Harmony Theory (HTML + PDF)
