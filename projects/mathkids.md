# MathKids v1.0

Aplicación web educativa para que niños de 7 años aprendan matemáticas jugando. Diseño colorido con Comic Neue, animaciones de recompensa y sistema de puntos.

## Features

- **5 módulos**: Sumas, Restas, Contar, Comparar, Formas
- **Gamificación**: Puntos, rachas, niveles
- **Feedback espectacular**: Confeti, caramelos, estrellas
- **Persistencia**: localStorage para progreso
- **Responsive**: Funciona en tablet y móvil

## Stack técnico

- **Frontend:** HTML5 + CSS3 + JavaScript vanilla
- **Fuente:** Comic Neue (Google Fonts)
- **Animaciones:** CSS animations + Canvas
- **Storage:** localStorage

## Aprendizajes clave

### Lo que funcionó bien

1. Comic Neue como fuente amigable para niños
2. Colores vibrantes con paleta consistente
3. Feedback inmediato y positivo

### Métricas

- **Módulos:** 5
- **Target:** Niños 7 años
- **Dependencias:** 0 (vanilla)
