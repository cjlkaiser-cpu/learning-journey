# AI Visual Lab

20 simulaciones interactivas de IA/ML con sonificacion + 20 guias cientificas + tutorial completo.

## Stack
- Canvas 2D para visualizaciones
- Web Audio API para sonificacion
- KaTeX para ecuaciones en guias
- Tailwind CSS en index
- GitHub Pages (gh-pages branch)

## Contenido
- **Modulo 1**: Perceptron, Gradient Descent, Backprop, Activation Functions, Weight Init
- **Modulo 2**: Autoencoder, LSTM, Attention, CNN, GAN
- **Modulo 3**: Transformer, Token Embeddings, Self-Attention, Chain of Thought, Emergent Abilities
- **Modulo 4**: Diffusion, RL, Knowledge Distillation, MoE, Superposition
